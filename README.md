# ericmurphyxyz/dotfiles

![Header Image](https://your-image-link-here.jpg)

This is where I keep all of my dotfiles.

Feel free to copy any of the config files or scripts you find here, as much of it is just copied from others' dotfiles. I wouldn’t recommend copying my entire setup, but instead try to gather bits and pieces as you put together your own dotfiles unique to you.

In the future, I plan on adding an install script to make it easier for others to use my config files. For now, you’ll have to piece things together yourself, though.

## Software I Use

If you’d like a more in-depth list of all the software and hardware I use on a daily basis, see the [Link page](https://your-link-here.com) on my website.

- **Operating System**: [Arch Linux](https://archlinux.org)
- **Window Manager**: [Hyprland](https://hyprland.org)
- **Status Bar**: [Waybar](https://github.com/Alexays/Waybar)
- **Terminal**: [Kitty](https://sw.kovidgoyal.net/kitty/)

### Where is everything?

Most config files for various programs can be found in the `config` directory. Shell scripts can be found in the `scripts` directory. Environment variables can be found in `.env`.

### Where’s all of your old config files?

I’m now using Wayland instead of Xorg. If you’re looking for old Xorg configs that may not be there anymore, you can take a look at the [Xorg branch](https://your-link-here.com) for my old, unmaintained Xorg config files.
